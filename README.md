Counter App

A minimal, single-page counter using local Storage - built with plain HTML, CSS, and JavaScript.

Features
- Increment, decrement, and reset the counter
- Small and simple

Usage
- Open [index.html](index.html) in your browser (or run with Live Server)
- Click the buttons to change the count

Files
- index.html — app UI
- script.js — counter logic
- style.css — styles

Notes
- Edit `script.js` to change behavior or add persistence.
- Free to use and modify.
